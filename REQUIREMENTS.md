# 小波海水浴場 Webサイト構築 要件定義書

## 1. プロジェクト概要
島根県松江市島根町にある「小波（こなみ）海水浴場」の公式Webサイトを構築プロジェクト。
透明度抜群の美しいビーチと併設されたキャンプ場の魅力を伝え、利用者の利便性を向上させることを目的としています。

## 2. 目的・ゴール
*   **魅力発信**: 透明度の高い海と遠浅のビーチ、キャンプ場の魅力を視覚的に訴求し、来場意欲を高める。
*   **情報集約**: アクセス、施設詳細、現地の天気など、利用者が知りたい情報を一元化する。
*   **アクション誘導**: キャンプ場の予約リクエストやお問い合わせをWeb上からスムーズに行えるようにする。

## 3. ターゲットユーザー
*   **ファミリー層**: 小さな子供連れでも安心して遊べる遠浅の海を探している家族。
*   **キャンパー**: 海辺のロケーションでキャンプを楽しみたいアウトドア愛好家。
*   **観光客**: 島根県松江市周辺の観光スポットを探している旅行者。

## 4. 機能要件

### 4.1 ページ構成 (Single Page Application)
Webサイトはシングルページアプリケーション（LP形式）として構成し、以下のセクションを含む。

1.  **ヘッダー (Header)**
    *   ロゴ表示
    *   ナビゲーションメニュー（ページ内スクロールリンク）
    *   SP版ではハンバーガーメニューを表示
    *   スクロール追従（Sticky）

2.  **ヒーローセクション (Hero)**
    *   ファーストビューでのインパクト重視のビジュアル
    *   キャッチコピー

3.  **ビーチ情報 (Beach Info)**
    *   小波海水浴場の特徴（透明度、遠浅、ファミリー向けなど）の紹介
    *   視覚的な魅力訴求

4.  **キャンプ場情報 (Camping Info)**
    *   キャンプサイトの概要
    *   施設・設備の説明

5.  **アクセス (Access)**
    *   Google Mapsによる地図表示
    *   住所、車でのアクセス方法
    *   「天気をチェック」ボタン（天気セクションへの誘導または外部サイトリンク）

6.  **天気情報 (Weather)**
    *   現地の天気予報表示（天気アイコン、気温、風速など）
    *   ※外部API連携またはモックアップ

7.  **フッター (Footer)**
    *   コピーライト
    *   基本情報

### 4.2 インタラクティブ機能・フォーム
*   **キャンプ場予約機能**
    *   「キャンプ場を予約」ボタンからモーダルウィンドウを展開。
    *   **入力項目**:
        *   お名前（漢字・カナ）
        *   チェックイン日・チェックアウト日
        *   住所
        *   電話番号
        *   メールアドレス
    *   **処理**: 入力内容をAPI経由で管理者のメールアドレスへ送信（Nodemailer使用）。

*   **お問い合わせ機能**
    *   「お問い合わせ」ボタンからモーダルウィンドウを展開。
    *   **入力項目**:
        *   お名前
        *   メールアドレス
        *   お問い合わせ内容
    *   **処理**: 入力内容をAPI経由で管理者のメールアドレスへ送信。

## 5. 非機能要件

### 5.1 デザイン (UI/UX)
*   **コンセプト**: 「Premium」「Modern」「Dynamic」。
*   **配色**: 海の青 (`#0077be`)、砂浜のベージュ (`#f4e4bc`) を基調とした、自然と調和するカラーパレット。
*   **スタイリング**: Tailwind CSS v4 を採用し、カスタムテーマ (`@theme`) で一貫性を管理。
*   **レスポンシブ**: モバイルファーストを意識し、スマートフォン、タブレット、デスクトップすべてのデバイスで最適化された表示。

### 5.2 技術スタック
*   **フレームワーク**: Next.js 16 (App Router)
*   **言語**: TypeScript
*   **スタイリング**: Tailwind CSS v4
*   **メール送信**: Nodemailer
*   **アイコン**: Lucide React
*   **デプロイ環境**: Vercel

### 5.3 パフォーマンス・SEO
*   Next.js の機能を活用した高速なページロード。
*   適切なメタタグ（Title, Description）の設定によるSEO対策。
*   セマンティックなHTML構造。

## 6. 今後の拡張ロードマップ
*   **Googleカレンダー連携**: 予約状況のリアルタイム表示と管理の自動化。
*   **多言語対応**: インバウンド需要への対応（英語、中国語など）。
*   **CMS導入**: お知らせやブログ機能の追加。
